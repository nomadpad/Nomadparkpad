Nomad Park Pad v3.0

UPLOAD TO GITHUB
1. Unzip this package.
2. Open the main level of your Nomad Park Pad GitHub repository.
3. Upload and replace:
   - index.html
   - styles.css
   - script.js
4. Upload the complete assets folder.
5. Commit with:
   Launch Nomad Park Pad v3

Vercel should redeploy automatically.

Important:
The Early Access form is visual only. It does not yet save emails.
