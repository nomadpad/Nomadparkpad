UPLOAD ON IPHONE
1. Download and unzip Phase5.zip in the Files app.
2. Open the main level of the GitHub repository.
3. Tap Add file, then Upload files.
4. Select every file inside the unzipped folder.
5. Replace matching files.
6. Commit with:
   Launch Phase 5 host dashboard
