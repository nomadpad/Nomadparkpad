NOMAD PARK PAD PHASE 2 — EASY GITHUB UPLOAD

Every file in this package belongs in the MAIN level of your GitHub repository.
There is no assets folder in this version.

ON IPHONE:
1. Download and unzip this package in the Files app.
2. Open your GitHub repository.
3. Tap Add file / Upload files.
4. Select every file inside this folder.
5. Upload them to the main branch.
6. Replace matching files when GitHub asks.
7. Commit with:
   Launch Phase 2 easy upload
